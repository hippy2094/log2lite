Import apache log files into sqlite3 database
(c)2016 Matthew Hipkin <http://www.matthewhipkin.co.uk>

Requires FreePascal to build http://www.freepascal.org

Uses BRRE https://github.com/BeRo1985/brre

Build with:

# fpc -O3 -Xs log2lite.pas

Simply run the binary with the log file as the only parameter
